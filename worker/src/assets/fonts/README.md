# Worker fonts

These font files are bundled into the Cloudflare Worker for server-rendered Store imagery.

- `inter-roman.bin` and `inter-italic.bin` are Inter font binaries covered by `LICENSE.inter.txt`.
- `gambado-sans-regular.bin` and `gambado-sans-forte.bin` are deployment-provided Gambado Sans font binaries used for display typography. Only deploy or redistribute them when your project has the appropriate font license.
